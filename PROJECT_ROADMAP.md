## P1 · 종목 카드 스와이프 (발견 척추 Step 1)

- **status**: active
- **milestone**: 발견 척추
- **okr**: 유저가 종목 카드를 스와이프해서 관심 종목을 저장할 수 있다
- **why**: PRODUCT_VISION §11 빌드 순서 1번. 여기가 없으면 나머지 모두 의미 없음. 제품의 심장.
- **success**: apps/fomo-web에서 포모 점수가 박힌 종목 카드를 스와이프(넘김/♥)할 수 있고, ♥한 카드가 히스토리에 저장된다.
- **scope**: 종목 카드 컴포넌트(포모 점수 + 💎 배지 + 사실 한 줄), 스와이프 인터랙션, 익명 세션 기반 ♥ 저장
- **constraints**: 포모 점수는 카드 안 온도계로만(독립 랭킹 화면 금지). 💎는 사실까지만(예측 금지). 카드는 종목 전용(테마·매크로 카드 섞기 금지).
- **breakdown**: 백엔드(Codex): 종목 카드 데이터 API (`/api/fomo/cards`) — 포모 점수 + 💎 여부 + 사실 한 줄 반환 / 프론트(Claude Code): 카드 컴포넌트 + 스와이프 UX + 익명 세션 ♥ 저장
