# FOMO Index — 체감 온도계

> ⚠️ **[폐기 — 히스토리 보존]** 이 문서의 정체성("감정 체감 온도계"·감정 시각화)은 **현재 기준이 아니다.** 현재 정본: `docs/PRODUCT_VISION.md` (**주식시장의 틴더 v5 — 두 표면·한 엔진**). 포모 점수는 **발견의 연료**(쏠림 온도)일 뿐 독립 진열 상품이 아니다. (단, 아래 점수 5구간·색 매핑은 발견 엔진이 계속 참조 — `KEYWORD_ENGINE_SPEC`.) 새 작업·에이전트는 SSOT(PRODUCT_VISION)를 따른다.

## 정의
투자자 집단 심리의 체감 온도계. **사람을 측정하는 지표이지 금융 지표가 아니다.**
목표: 앱을 열었을 때 "오늘 시장 분위기 어떻지?"를 3초 안에 이해하게 만든다.

- 범위: 0 ~ 100
- 산출 주체: `packages/fomo-core` (산출 로직), 일일 파이프라인이 스냅샷 저장
- 면책: FOMO Index는 금융 지표가 아닌 감정 체감 지표이며, 투자 조언이 아니다.

## 스케일 (5구간)

| FOMO Index | 상태 | 의미 |
|---|---|---|
| 0~20 | 😴 무관심 | 시장이 조용하다 |
| 21~40 | 🙂 관망 | 관심은 있지만 적극적이지 않다 |
| 41~60 | 👀 관심 | 특정 종목/섹터에 관심 집중 |
| 61~80 | 🔥 FOMO | 놓치기 싫은 심리 증가, SNS/커뮤니티 급증 |
| 81~100 | 🚀 광기 | 집단 심리 과열, 시장보다 감정이 앞섬 |

## 공식

```
FOMO Index = Market Heat(30%) + Community Heat(30%) + Emotion Heat(30%) + Whale Heat(10%)
```

### 1. Market Heat (0~30)
거래량 급증, 거래대금 증가, 주요 키워드 검색량, ETF 자금 유입.

### 2. Community Heat (0~30)
X/Reddit/Stocktwits 언급량·게시물·댓글 변화율.
- Bullish 키워드: To The Moon, All In, Full Position, Diamond Hands, YOLO
- Bearish 키워드: 청산, 손절, 망했다, 물렸다

### 3. Emotion Heat (0~30)
FOMO Club 사용자 감정 투표 집계. FOMO/탐욕 비중↑ → 상승, 공포/후회↑ → 하락.

### 4. Whale Heat (0~10)
이벤트 기반: 대형 청산 위기, ETF 대규모 유입, BTC 신고가, 섹터 급등, Short Squeeze.

> 각 Heat는 데이터 소스 미비 시 안전한 기본값 + 폴백으로 산출한다. 절대 빈 값/에러를 노출하지 않는다.

## 홈 표시 형식

```
FOMO INDEX 74 🔥 / 현재 상태 FOMO / 전일 대비 +8 / 30일 평균 대비 +17
```
- AI Summary: 오늘 시장 감정 1~2문장 요약
- Daily Insights: 롤링 배너, 🚨 형식 (예: "오늘 FOMO 지수 80 돌파", "AI 언급량 +240%")

## FOMO Index ↔ 마스코트 포모 표정 매핑

5구간은 마스코트 '포모'의 5표정과 직결된다(docs/MASCOT.md §5 1단계). **구간 경계값을 변경하면 표정 매핑도 함께 점검해야 한다.**

| FOMO Index | 상태 | 포모의 표정 |
|---|---|---|
| 0~20 | 😴 무관심 | 졸린 듯 무관심한 얼굴, 눈이 반쯤 감김 |
| 21~40 | 🙂 관망 | 차분히 지켜보는 얼굴 |
| 41~60 | 👀 관심 | 두리번거리며 관심 갖는 얼굴, 눈이 또렷 |
| 61~80 | 🔥 FOMO | 눈이 커지고 빨갛게 달아오른 들뜬 얼굴 |
| 81~100 | 🚀 광기 | 거의 광기에 가까운 흥분한 얼굴 |
